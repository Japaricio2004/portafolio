Toque Mágico – Juego Web Interactivo

Juego web casual que pone a prueba la velocidad y reflejos del usuario al destruir orbes mágicas en 30 segundos. Interfaz dinámica, efectos visuales llamativos y estilo arcade. Optimizado para navegadores modernos.

🎯 Características principales:

  • Juego de velocidad y reflejos
  • Efectos visuales llamativos
  • Estilo arcade moderno
HTML5 Tailwind CSS JavaScript Google Fonts

Agenda Interactiva 3D – Aplicación Web de Calendario Personal

Aplicación web moderna y responsiva que permite gestionar tareas diarias con una interfaz visual atractiva y animaciones tipo 3D. Incluye calendario interactivo, modo claro/oscuro con persistencia, y está optimizada para todo tipo de dispositivos.

✨ Características principales:

  • Calendario interactivo con animaciones 3D
  • Modo claro/oscuro con persistencia
  • Sin necesidad de base de datos
HTML5 JavaScript Tailwind CSS LocalStorage

LivePersona – Simulador Interactivo de Personalidad

Aplicación web educativa que simula cómo distintos rasgos psicológicos (introversión, empatía, impulsividad) pueden influir en el comportamiento de un personaje virtual. Ofrece controles deslizantes para ajustar valores y ver resultados en tiempo real.

🎮 Características principales:

  • Simulación de rasgos psicológicos
  • Controls deslizantes interactivos
  • Resultados en tiempo real
HTML5 Tailwind CSS JavaScript
🌐

Desarrollo Web Responsivo – Bodega M.A

Sitio web orientado a mejorar la presencia digital de tiendas pequeñas, con enfoque en accesibilidad, diseño moderno y compatibilidad móvil. Incluye navegación intuitiva, secciones informativas, estética elegante, y fue desarrollado con HTML5, Tailwind CSS y JavaScript básico.

✨ Características principales:

  • Navegación intuitiva y clara
  • Integraciones con WhatsApp y Google Maps
  • Diseño amigable y adaptable
HTML5 Tailwind CSS JavaScript API WhatsApp API Google Maps

Plataforma de Gestión de Reservas

Plataforma web adaptable para pequeños negocios y profesionales independientes, diseñada para gestionar reservas en línea de manera eficiente. Los clientes pueden registrarse y acceder al sistema de forma sencilla y sin complicaciones.

🗂️ Características principales:

  • Ficha personalizada por negocio
  • Formulario de reserva accesible
  • Diseño amigable y adaptable
HTML5 JavaScript Python Flask SQLite
🌎

SaludPlus – Bienestar Personal

SaludPlus es una aplicación web innovadora diseñada para ayudarte a llevar un seguimiento detallado y eficiente de tu bienestar personal. Con SaludPlus, puedes gestionar métricas clave como peso, sueño, actividad física, y otros aspectos de tu salud de manera sencilla y visual.

🩺 Características principales:

  • Seguimiento de métricas de salud (peso, sueño, actividad física, etc.)
  • Visualización sencilla y atractiva de datos
  • Gestión eficiente y segura de la información personal
HTML5 JavaScript Python Flask SQLite
